### 名詞解釋

當我們在說 Content-Type 跟 MIME Type 或是 Media Type 的時候，其實通常都是在說同一件事情，差別在於:

1. Content-Type 是一個 HTTP Header，其值就是 MIME Type

2. MIME Type 是比較早期的說法，後改為 Media Type，但兩個其實大家都聽得懂
